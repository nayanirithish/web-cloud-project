# WebCloud - Secure Cloud Repository System

## 📌 Description
WebCloud is a secure cloud-based repository system built using Java, JSP, and Servlets with MySQL. It ensures protected data sharing using encryption and key-based access control.

## 🚀 Technologies Used
- Java
- JSP (Java Server Pages)
- Servlets
- MySQL
- HTML, CSS, Bootstrap
- JavaScript

## 🔐 Security Features
- AES / Base64 Encryption
- SHA1 Integrity Verification
- Key-based Access Control


## ⚙️ Features
- Secure file upload and download
- User authentication system
- Data integrity verification
- Controlled access using keys

## ▶️ How to Run
1. Install Apache Tomcat
2. Import project into Eclipse
3. Configure MySQL database
4. Run on server
